HTML5 tiene un conjunto de atributos de validación que puedes usar directamente en los elementos del formulario para asegurarte de que los datos son correctos antes de enviarlos. Ejemplo de formulario con validación en HTML5: Validación de Formulario

Formulario de Registro









## Explicación: #### Validación HTML5: required: Hace que el campo sea obligatorio. minlength y maxlength: Establecen un rango de longitud para el campo de texto. type="email": Verifica que el valor ingresado tenga un formato de correo electrónico. type="password": Marca el campo como un campo de contraseña, lo que oculta el texto mientras lo escribe el usuario. #### Validación con JavaScript: Se agrega un eventListener al formulario para interceptar el envío (submit). Antes de enviar los datos, se valida que las contraseñas coincidan. Si las contraseñas no coinciden, se muestra un mensaje de alerta y se evita que el formulario sea enviado (event.preventDefault()).
