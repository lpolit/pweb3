Formulario de Registro









## Explicación del código: #### Validación Manual: Se valida el nombre de usuario (mínimo 5 caracteres). Se usa una expresión regular para validar el correo electrónico. Se valida que las contraseñas coincidan y tengan al menos 8 caracteres. #### Prevención de Envío: Si algún campo no es válido, se muestra un mensaje de alerta y se previene el envío del formulario usando event.preventDefault().
